# Homebrew Tap for Obsi

Homebrew formula for [obsi](https://github.com/avi8or/obsidian-cli-quick-notes) â€” a CLI for Obsidian note-taking (Quick Notes Fork).

## Installation

```bash
brew tap avi8or/obsi
brew install obsi
```

## Usage

```bash
obsi --help
obsi init          # Interactive setup wizard
obsi append "note" # Quick append to daily note
obsi todo "task"   # Use named targets
```

See the [main repo](https://github.com/avi8or/obsidian-cli-quick-notes) for full documentation.
